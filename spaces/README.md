# DnD Offline Pro — Web Demo

This Space is a live preview of [DnD Offline Pro](https://github.com/gokulrajanpillai/DnDOfflinePro),
a fully offline AI dungeon master that runs on any CPU with no internet connection.

The demo runs the same model (Qwen2.5-0.5B-Instruct) in the cloud so you can try the experience
before downloading. The downloadable version runs 100% on your own machine — no API key, no
subscription, no data sent anywhere.

**[Download the offline version on GitHub](https://github.com/gokulrajanpillai/DnDOfflinePro)**
